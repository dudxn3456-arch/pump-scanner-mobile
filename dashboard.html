설정 순서

처음 한 번만 권한 승인을 완료하면, 이후에는 PC가 꺼져 있어도 Google Apps Script가 실행됩니다.

  1. Google Sheet Binance USDT-M 급등주 스캐너 Cloud를 엽니다.
  2. 상단 메뉴에서 확장 프로그램 > Apps Script를 엽니다.
  3. Drive에 있는 Binance_USDTM_Pump_Scanner_Code.gs의 전체 내용을 복사합니다.
  4. Apps Script 편집기의 기본 Code.gs 내용을 지운 뒤 붙여넣고 저장합니다.
  5. 함수 선택 드롭다운에서 setupTrigger를 선택하고 실행합니다.
  6. 처음 1회 권한 승인을 완료한 뒤 Summary와 Scanner 탭을 확인합니다.

판정 기준 요약

A_GRADE
거래대금, 거래량, 거래 횟수 증가가 강하고 종가 위치가 높으며 윗꼬리 부담이 제한적인 상위 후보입니다.
PUMP_CANDIDATE
급등 후보. 거래대금과 가격 움직임이 커졌지만 추가 확인이 필요한 단계입니다.
WATCH
거래량이 예열되는 관찰 단계입니다. 추격 진입보다 구조 확인이 우선입니다.
DIST_RISK
고점 대비 종가 위치가 낮거나 윗꼬리가 커서 분배 위험으로 분류된 단계입니다.
실행 함수
setupTrigger(): 15분 자동 실행 트리거를 생성하고 즉시 1회 스캔합니다.
수동 실행
scanBinanceUsdtmPumpCandidates(): 현재 시점 기준으로 한 번 스캔합니다.
중지
removeScannerTriggers_(): 자동 스캔 트리거를 삭제합니다.
주의
정보 제공용 스캐너이며 자동매매, 매수/매도 권유, 수익 보장을 의미하지 않습니다.

# Binance USDT-M Pump Scanner Cloud

This folder contains the Google Apps Script version of the Binance USDT-M pump scanner.

Google Drive is storage, not a compute runtime. To keep scanning while the PC is off,
use a Google Sheet with a bound Apps Script time trigger.

Setup:
1. Open the Google Sheet named `Binance USDT-M Pump Scanner Cloud`.
2. Go to Extensions > Apps Script.
3. Paste the full contents of `Code.gs`.
4. Save the project.
5. Run `setupTrigger()` once.
6. Approve permissions.

After setup, Google Apps Script runs `scanBinanceUsdtmPumpCandidates()` every 15 minutes
and writes results to:
- `Summary`
- `Scanner`
- `AllData`
- `Errors`
- `DetectionLog`

Notes:
- The scanner uses Binance Futures public API data.
- `DetectionLog` keeps each symbol's first detected time, latest detected time, stage changes, and detection count.
- The scanner is informational only and is not an automated trading signal.
- If API errors or timeouts occur, check the `Errors` sheet.
